# NIK.DEV — портфолио разработчика

Этот проект — мой персональный сайт‑портфолио.  
Он создан для практики верстки, адаптивного интерфейса, работы с изображениями и организации небольшого фронтенд‑проекта без фреймворков.

---

## Основные возможности

- Адаптивная верстка (mobile‑first) 
- Многоуровневая архитектура CSS 
- Design Tokens (цвета, размеры, типографика) 
- Простое мобильное меню на JavaScript 
- Модальное окно с просмотром изображений 
- SVG‑спрайт для иконок 
- Локальные шрифты Montserrat (cyrillic_latin) 
- Легкая оптимизация под Lighthouse

## CSS Architecture

Проект использует многоуровневую архитектуру CSS, вдохновлённую ITCSS/SMACSS и принципами дизайн‑систем.  
Каждый файл структурирован по уровням, что делает код предсказуемым, читаемым и удобным для поддержки.

### Уровни архитектуры

```md
### Уровни архитектуры
```

Каждый CSS‑файл содержит чёткие разделы:

```css
/* ============================================================
   УРОВЕНЬ X: НАЗВАНИЕ
   ============================================================ */
```

Используются следующие уровни:

1. **Контейнер секции** — базовая геометрия и фон  
2. **Контейнер контента** — layout и внутренние блоки  
3. **Медиа‑элементы** — изображения, иконки  
4. **Текстовые блоки** — заголовки, параграфы, статусы  
5. **Элементы управления** — кнопки, ссылки  
6. **Состояния и анимации** — hover, active, keyframes  
7. **Адаптивные стили** — desktop, tablet, prefers‑reduced‑motion

## Design Tokens

Все цвета, размеры, отступы, радиусы, тени и типографика вынесены в `:root`:

- `--bg-*` — фоновые слои  
- `--text-*` — текстовые цвета  
- `--interactive-*` — интерактивные элементы  
- `--spacing-*` — отступы  
- `--font-size-*` — типографика  
- `--radius-*` — скругления  
- `--z-*` — уровни наложения
